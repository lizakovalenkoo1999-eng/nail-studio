## Тренды дизайна

### Основные принципы (3 столпа)

**1. Доверие (Trust):**
- Визуальное портфолио высокого качества
- Отзывы клиентов
- Секция "Нас рекомендуют" (featured in)
- Раздел "О нас" с историей салона

**2. Информация (Information):**
- Чёткое описание услуг с ценами
- Система онлайн-записи
- Информация о местоположении и парковке
- Контактные данные

**3. Дизайн (Design):**
- Соответствие бренду салона
- Современный и профессиональный вид
- Адаптивность под мобильные устройства

## Цветовые палитры

### Для люксовых spa-салонов
- **Основные цвета:** мягкие нейтральные тона
  - Слоновая кость (Ivory)
  - Золото (Gold)
  - Нежно-розовый (Blush pink)
- **Шрифты:** элегантные с засечками (Playfair Display, Garamond)

### Для современных трендовых nail-bar
- **Основные цвета:** яркие и смелые
  - Чёрный (Black)
  - Ярко-розовый (Hot pink)
  - Металлик (Metallic accents)
- **Шрифты:** дерзкие (Montserrat, Bebas Neue)

### Для натуральных/эко салонов
- **Основные цвета:** спокойные природные
  - Мягкий зелёный
  - Пастельные коричневые
  - Земляные тона
- **Шрифты:** чистые sans-serif (Open Sans, Lato)

### Рекомендуемая палитра для сайта
```css
:root {
  /* Основные цвета */
  --primary: #D4A5A5;        /* Розовый nude */
  --primary-dark: #B88888;   /* Тёмный розовый */
  --secondary: #C9A96E;      /* Золотистый */
  --accent: #E8D5D5;         /* Светлый розовый */

  /* Нейтральные */
  --bg-light: #F8E8EE;       /* Очень светлый розовый */
  --bg-white: #FFFFFF;
  --text-dark: #2D2D2D;
  --text-light: #666666;

  /* Акцентные */
  --success: #4CAF50;
  --gold: #C9A96E;
}
```

## Типографика

### Рекомендуемые шрифты

**Заголовки (Heading):**
- `Playfair Display` - элегантный для люксового сегмента
- `Montserrat` - современный и чистый
- `Bebas Neue` - для акцентов и крупных заголовков
- `Cormorant Garamond` - классический и изысканный

**Основной текст (Body):**
- `Lato` - отличная читаемость
- `Open Sans` - универсальный
- `Montserrat` - современный
- `Inter` - для интерфейсов

**Рекомендуемая связка:**
```css
/* Люксовый стиль */
--font-heading: 'Playfair Display', serif;
--font-body: 'Lato', sans-serif;

/* Современный стиль */
--font-heading: 'Montserrat', sans-serif;
--font-body: 'Open Sans', sans-serif;
```

## Анимации и эффекты

### 1. Анимации при скролле (Scroll Animations)
- **Fade-in** - плавное появление элементов при скролле
- **Slide-in** - выезд элементов сбоку/снизу
- **Stagger** - последовательное появление элементов списка
- **Parallax** - параллакс-эффект для фонов

### 2. Hero Section
- **Видео-фон** -背景 видео с затемнением
- **Particle effects** - частицы (элегантно для spa)
- **Ken Burns** - медленное масштабирование изображения
- **Text reveal** - анимация появления текста по буквам/словам

### 3. Интерактивные элементы
- **Hover effects на карточках:**
  - Подъём (transform: translateY)
  - Тень (box-shadow)
  - Масштабирование изображения (scale)
  - Изменение цвета фона

- **Кнопки:**
  - Плавное изменение цвета
  - Анимация иконки (стрелка, телефон)
  - Ripple effect

### 4. Галерея
- **Lightbox** - модальное окно с изображением
- **Carousel** - карусель с миниатюрами
- **Before/After** - слайдер сравнения
- **Masonry grid** - плиточная сетка разного размера

### 5. Формы и UI
- **Floating labels** - всплывающие подписи полей
- **Input animations** - анимация фокуса
- **Loading states** - состояния загрузки
- **Success animations** - анимация успешной отправки

### 6. Специфичные эффекты для nail-индустрии
- **Liquid/ink effects** - эффект чернил
- **Glitter** - блёстки (праздничные дизайны)
- **Shimmer** - мерцание
- **3D nail preview** - 3D просмотр дизайнов ногтей

### Примеры CSS-анимаций
```css
/* Появление при скролле */
@keyframes fadeInUp {
  from {
    opacity: 0;
    transform: translateY(30px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}

/* Плавный hover на карточку */
.card {
  transition: all 0.3s ease;
}
.card:hover {
  transform: translateY(-8px);
  box-shadow: 0 20px 40px rgba(0,0,0,0.1);
}

/* Анимация кнопки */
.btn {
  position: relative;
  overflow: hidden;
  transition: all 0.3s ease;
}
.btn::after {
  content: '';
  position: absolute;
  width: 100%;
  height: 100%;
  background: rgba(255,255,255,0.3);
  transform: scale(0);
  transition: transform 0.5s ease;
}
.btn:active::after {
  transform: scale(2);
}
```

## Ключевые разделы сайта

### 1. Hero-секция
- Привлекательный заголовок (слоган)
- Подзаголовок с описанием
- CTA-кнопка "Записаться"
- Фоновое изображение/видео
- Декоративные элементы

### 2. Преимущества (Advantages)
- Иконки + текст
- 4-6 ключевых преимуществ
- Анимация при скролле

### 3. Услуги (Services)
- Табы для категорий (маникюр, педикюр, наращивание, дизайн)
- Карточки услуг с ценами
- Описание каждой услуги
- Кнопка записи на услугу

### 4. Галерея работ (Portfolio)
- Сетка изображений (masonry)
- Фильтры по категориям
- Lightbox при клике
- Hover-эффекты

### 5. О салоне (About)
- История салона
- Философия
- Фото интерьера
- Счётчики (клиенты, годы, мастера)

### 6. Мастера (Team)
- Фото мастеров
- Имя и специализация
- Опыт работы
- Соцсети
- Кнопка записи к конкретному мастеру

### 7. Отзывы (Testimonials)
- Слайдер/карусель
- Фото клиентов
- Текст отзыва
- Рейтинг (звёзды)
- Дата

### 8. Онлайн-запись (Booking)
- Форма записи
- Выбор услуги
- Выбор мастера (опционально)
- Дата и время
- Контактные данные
- Комментарий

### 9. Контакты (Contact)
- Адрес на карте
- Телефон
- Email
- Время работы
- Соцсети
- Форма обратной связи

### 10. FAQ (Часто задаваемые вопросы)
- Аккордеон с вопросами-ответами
- Помогает снизить нагрузку на поддержку

## Онлайн-запись и функционал

### Система онлайн-записи
- Выбор услуги из списка
- Выбор мастера (опционально)
- Выбор даты и времени
- Автоматическое подтверждение
- SMS/Email напоминания
- Календарь доступности в реальном времени

### Дополнительный функционал
- **Конструктор дизайна** - выбор цвета, формы, декора
- **Стоимость услуг** - калькулятор
- **Подарочные сертификаты** - покупка онлайн
- **Программа лояльности** - бонусы для клиентов
- **Блог** - статьи об уходе за ногтями
- **Instagram feed** - интеграция с соцсетями
- **Chatbot** - для быстрых вопросов

## Технические рекомендации

### Производительность
- **Изображения:** WebP формат, lazy loading
- **Минификация:** CSS, JS
- **Кэширование:** статические ресурсы
- **CDN:** для быстрой загрузки

### Адаптивность (Mobile-first)
- Мобильная версия приоритет
- Touch-friendly элементы (min 44px)
- Hamburger-меню для мобильных
- Адаптивные сетки

### SEO
- Semantic HTML
- Meta-теги
- Schema.org для локального бизнеса
- Open Graph для соцсетей
- Sitemap и robots.txt

### Доступность (Accessibility)
- Alt-тексты для изображений
- ARIA-атрибуты
- Keyboard navigation
- Color contrast (WCAG)
- Focus states

### Безопасность
- HTTPS
- Защита форм от спама
- Валидация данных
- Безопасная обработка платежей
